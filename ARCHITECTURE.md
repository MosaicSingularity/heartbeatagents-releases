# Architecture

This repository carries the signed artifacts that make HeartBeatAgents
run on a customer's machine — the installers, checksums, and update
manifests the in-app updater consumes. It is not where the architecture
itself is documented.

For that, we maintain a single source of truth: the HeartBeatAgents
Blueprint. It traces every layer end to end — the desktop client, the
container runtime, the agent execution model, and the architectural
decisions that connect them — at the depth our engineering and security
teams expect.

**→ [The HeartBeatAgents Blueprint](https://www.heartbeatagents.com/blueprint)**
